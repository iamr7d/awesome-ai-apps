# Awesome AI Apps & Agents

A curated collection of AI applications, starter kits, and tutorials for building intelligent agents.

## Contents

- **Starter AI Agents**: Boilerplate code and examples for building agents.
- **Pydantic AI Tutorials**: Guides on using Pydantic AI for structured outputs and validation.

## Getting Started

Explore the `starter_ai_agents` directory for specific implementation examples.
Most projects require Python 3.9+.

```bash
cd starter_ai_agents/pydantic_ai_tutorial
pip install -r requirements.txt
```
